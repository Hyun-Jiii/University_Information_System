# 대학 정보 시스템 설계
동의대학교 컴퓨터소프트웨어공학과 이종민 교수님의 설계공학 과목인 '객체지향프로그래밍Ⅱ' 수업 중 *대학 정보 시스템 설계 구현 프로젝트* 입니다.

# 문제 기술서
대학은  대학  구성원인  교수, 학생, 직원의  작업을  효율적으로  지원하기  위한  전산  시스템을  구축하려고  한다. 전산  시스템은  크게  학사  관리  기능, 수업  관리  기능, 수강 관리  기능 사용자  관리  기능을  제공해야  한다.
